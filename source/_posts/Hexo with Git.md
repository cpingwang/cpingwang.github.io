---
title: Hexo with Git
---


Hexo的指令

    $ hexo clean
    
    $ hexo generate
    
    發佈文章:
    $ hexo deploy
    
    $ hexo server
    
    More info: [Hexo](https://hexo.io/zh-tw/docs/writing.html)
---
Git的指令

    將異動加入Git:
    $ git add .
    
    Commit本地並加上注解說明:
    $ git commit -a -m "comments"
    
    將Commit的內容推送到Git:
    $ git push origin hexo
